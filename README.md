# 📚 Chuletazos de 0 a PRO

Repositorio con chuletazos técnicos para repasar y practicar comandos y flujos de trabajo en distintos entornos.  
Cada archivo `.md` contiene un resumen práctico, con ejemplos comentados, pensado para consulta rápida.

## 📂 Contenido

- **bash.md** → Comandos Bash, regex, `grep`, `awk`, `sed`, backup y compresión.
- **git.md** → Flujo Git y GitHub: inicializar, conectar, actualizar, ramas, `pull` y `push`.
- **docker.md** → Contenedores, imágenes, volúmenes, redes y buenas prácticas.
- **laravel.md** → Migraciones, seeders, factories, CRUD y manejo de archivos.
- **sql.md** → Consultas `JOIN`, `HAVING`, `SUM`, vistas, funciones, procedimientos y triggers.
